### 实验内容

#### Task 1：语音特征提取

* 读取语音数据
* 提取 MFCC特征

#### Task 2：基于 DTW 的语音匹配与识别

* 实现 DTW 算法，用于计算两个语音样本之间的相似度
* 在给定的语音样本集上，利用 DTW 进行模板匹配识别
* 评估识别准确率、混淆矩阵等指标

**可选改进：**

* 在 DTW 基础上尝试使用加速方法（如 Sakoe-Chiba band、FastDTW）
* 比较不同距离度量方式（欧氏距离、余弦距离等）的效果差异

#### Task 3：实验对比与分析

* 对比不同特征类型或不同 DTW 参数设置下的识别准确率
* 分析 DTW 在语音识别中的优势与局限性（如计算量、对噪声的敏感性等）
* 若尝试了改进方法（如降维、模板平均、动态带宽限制等），请说明其效果与原因

### 结果展示与分析
